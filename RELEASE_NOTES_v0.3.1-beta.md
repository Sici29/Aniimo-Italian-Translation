# Aniimo — Traduzione italiana v0.3.1 Beta

Questa release completa il nuovo pannello intelligente dell'installer.

## Novità

- indica se Aniimo è stato trovato automaticamente oppure tramite un percorso salvato;
- controlla le risorse del gioco e mostra se la traduzione italiana è già installata;
- se il gioco non viene trovato, apre una finestra per scegliere facilmente la cartella;
- permette anche di incollare manualmente il percorso;
- spiega come riconoscere la cartella corretta: deve contenere `Aniimo_Data`;
- salva il percorso verificato per gli avvii successivi;
- aggiunge l'opzione `4` per modificare il percorso in qualsiasi momento.

## Installazione

Scarica e apri il solo file `Aniimo-Italian-Translation.exe`. Se possiedi già la versione 0.3.0 beta, l'aggiornamento può essere applicato automaticamente dall'installer.

Chiudi Aniimo e il launcher prima di installare o ripristinare la traduzione. Nel gioco continua a essere necessario selezionare **English**.
